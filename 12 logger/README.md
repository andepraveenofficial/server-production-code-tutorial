# Logger

### Techstack

1. Environment : `NodeJS`
2. Language : `Typescript`
3. Server : `express`
4. ORM : `prismaORM`
5. Database : `Postgresql`

### Logger Setup

1. Install winston : `npm install winston` => Logging library
2. Install morgan : `npm install morgan`
   - Install morgan types : `npm install -D @types/morgan`

### Installation

- `npm install`

### Start the Server

- Prisma Format : `npx prisma format`
- Connect prismaClient : `npx prisma generate`
- Migrations : `npm run migrate`
- Seeds : `npm run seed`
- Build : `npm run build`
- Production : `npm start`
- Development : `npm run dev`
- Prisma Studio (Database) : `npx prisma studio`
